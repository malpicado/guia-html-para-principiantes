Etiquetas básicas de HTML

Vamos a mencionar las etiquetas básicas de HTML que te recomiendo conocer, serán las que utilizarás a diario

Etiqueta <!DOCTYPE>

Esta etiqueta hace referencia al tipo de documento que estaremos trabajando en nuestro editor de texto.

Se tiene que indicar el tipo de documento al inicio de cada nuevo archivo en el IDE.

Se escribe así<!DOCTYPE>

Etiqueta <html>

Con esta etiqueta llamamos a la estructura HTML de nuestro archivo, donde añadiremos el head y el body de toda página web.

Se escribe así<html></html>

Etiqueta <head>

En esta etiqueta incluimos la información de cabecera de la página que estamos construyento.
Aquí mencionaremos a la etiqueta meta para definir el standar de codificación de la página.

Se escribe así 
                    <head>
                        <meta charset="utf-8">
                    </head>

Etiqueta <body>

Esta etiqueta es la que incluirá todo el contenido que tu visitanta visualizará en la página web.
Esta etiqueta contendrá las etiquetas de sección, títutlos y párrafos para dar estructura al contenido que se muestra.

Se escribe así 
                    <body>
                        <section>
                            <h1></h1>
                            <p></p>
                        </section>
                    </body>
