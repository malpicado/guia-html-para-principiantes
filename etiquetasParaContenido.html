Etiquetas para contenido

En esta sección, hablaremos de las principales etiquetas que utilizarás al momento de crear el contenido de tu página web.
En su mayoría, tendrás que incluirlas dentro de la etiqueta body.

Etiquetas de título

Hay varios tipos de etiquetas de título que utilizarás a diario, para darle jerarquía a la información que tu visitante visualizará en tu página web.

Las etiqueras van desde el "h1" hasta el "h6", pero es muy común que solo termines utilizando hasta el "h4".

Etiqueta <h1>

Esta etiqueta especifica el título principal de una página en especifico.
A modo de recomendación, solo debe existir un etiqueta h1 por cada url de tu página web, ya que es la de mayor jerarquía.

Se escribe así 
<h1>Título principal de esta página</h1>

Etiqueta <h2>

Esta etiqueta especifica el subtítulo o subtema principal, relacionado al título principal de una página en especifico.
Pueden haber varias etiquetas h2 en una sola página, ya que son la subcategoría de contenido más relevante.

Se escribe así 
<h2>Subtítulo principal relacionado con el título de la página</h2>

Etiqueta <h3>

Esta etiqueta especifica un tópico o subtema del subtítulo principal, normalmente sirve para mencionar contenido relacionado o que aporta algo de información al subtítulo donde está incluida.

Se escribe así 
<h3>Tópico que aporta algo al tema del subtítulo principal</h3>

Etiqueta <h4>

Esta etiqueta es de mayor profundidad de información, y normalmente se utiliza para mencionar algún contenido que está relacionado con el tópico de la etiqueta h3.

Se escribe así 
<h4>Soy un contenido relacionado con el tópico de la etiqueta h3</h4>

Etiquetas <h5> y <h6>

Estas etiquetas son las de profundidad máxima en la jerarquía de información, y se suelen utilizan para resaltar encabezados sobre temas muy segmentados o para ser modificados a través de CSS y que se vean diferentes a los textos de un párrafo.

Se escribe así 
<h5>Soy encabezado que puedes modificar con CSS</h5>
                    
<h6>Soy otro encabezado que puedes modificar con CSS</h6>

Etiqueta de párrafo

Esta estiqueta te ayudará a contener el texto que incluirás en tus diferentes páginas web.
Se usa como contenedor de texto plano.

Se escribe así 
<p>Soy un texto plano de tu contenido</p>
